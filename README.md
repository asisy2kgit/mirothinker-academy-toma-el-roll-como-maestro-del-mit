# Libro de carrera

Este repositorio contiene mi **libro de formación de carrera en programación**.

- Nivel inicial: preparatoria, sin experiencia formal.
- Nivel objetivo: llegar a un nivel equivalente a iniciar un doctorado en Ciencias de la Computación.
- Coordinador de carrera: asistente IA (ChatGPT), usando como referencia CS50x de Harvard y planes de estudio de universidades de la liga C9 y UNAM.

## Estructura del repositorio

- Carpeta `html-libro/`: páginas HTML coleccionables de cada unidad.
  - `unidad1.html`: resumen de la Unidad 1 (pensamiento computacional y Git).
